## ğŸ’¡ Key Takeaways

1. **Supabase Auth** handles semua autentikasi (register, login, password reset)
2. **RLS Policies** melindungi data di level database (user hanya bisa akses data mereka)
3. **Anon Key** aman digunakan di frontend karena dilindungi RLS
4. **Service Role Key** HANYA untuk Edge Functions atau backend (bypass RLS)
5. **JWT Token** disimpan otomatis di browser local storage oleh supabase-js
6. **Real-time** bisa diaktifkan untuk live updates (optional)
